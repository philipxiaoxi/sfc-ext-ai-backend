# WebSocket 协议更新：DONE 消息增强

## 更新日期

2026-07-11

## 变更说明

DONE 消息的 `data` 中新增以下字段，用于在 LLM 流响应完成时携带模型信息和调用耗时。

## 新增字段

DonePayload 新增字段：

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| `modelId` | `Long` | 否 | 模型 ID（数据库主键） |
| `time` | `Long` | 否 | 调用耗时（毫秒） |

## 示例

### 正常完成

```json
{
  "type": "DONE",
  "data": {
    "reason": "已完成",
    "modelId": 1,
    "time": 3521
  }
}
```

### 用户手动停止（无模型信息）

```json
{
  "type": "DONE",
  "data": {
    "reason": "已停止"
  }
}
```

## 兼容性说明

- 新增字段均为可选字段，不会影响现有客户端解析
- `modelId` / `time` 仅在 LLM 流正常完成时存在
- 用户手动停止（STOP 消息）或错误场景下，DONE/ERROR 消息可能不包含这些字段
